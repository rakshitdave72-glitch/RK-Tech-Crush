# R.K Tech Crush — Digital Agency Website

Premium, conversion-focused website for **R.K Tech Crush** — SEO, web development, WhatsApp marketing, LinkedIn marketing, and digital growth services.

## Quick Start

```bash
cd rk-tech-crush
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
npm run preview
```

## Project Structure

- `src/pages/Landing.tsx` — Home page with all sections
- `src/pages/PricingPage.tsx` — All service pricing in one place
- `src/pages/ServicePage.tsx` — Reusable service page template
- `src/data/` — Site config, pricing, and SEO content
- `public/images/` — Logo, banner, and founder photo

## Replace Founder Photo

Place your profile photo at `public/images/founder.jpg` (recommended: 600×750px).

## Features

- Dark neon tech design (electric blue / cyan)
- Transparent logo in navbar & footer
- 6 individual service pages with SEO content
- Pricing on each service page + dedicated `/pricing` page
- Contact forms on home and each service page
- Fully responsive layout
- Framer Motion animations
